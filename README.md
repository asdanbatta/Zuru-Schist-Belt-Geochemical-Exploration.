# Zuru Schist Belt Geochemical Exploration Using Python

## Overview

This repository presents a geochemical exploration study of the Zuru Schist Belt, Northwestern Nigeria, using published geochemical datasets and Python-based spatial analysis.

The project demonstrates modern data science techniques for mineral exploration, including statistical analysis, anomaly detection, spatial interpolation, and exploration target generation.

---

## Objectives

- Identify geochemical anomalies.
- Evaluate relationships among geochemical elements.
- Generate exploration targets.
- Demonstrate Python applications in economic geology.

---

## Dataset

This project uses published geochemical data from:

Fagbohun et al. (2021), Arabian Journal of Geosciences.

The dataset contains real field observations and laboratory geochemical analyses.

---

## Methods

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Descriptive Statistics
- Spearman Correlation Analysis
- Geochemical Anomaly Detection
- IDW Interpolation
- Multi-element Halo Analysis
- Exploration Target Ranking
